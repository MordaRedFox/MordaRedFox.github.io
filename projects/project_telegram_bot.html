Telegram Feedback Bot

📅 2025 🏷️ Python, python-telegram-bot, SQLite, asyncio 🎓 Учебный проект

Асинхронный Telegram-бот для сбора обратной связи от пользователей. Позволяет отправлять предложения, жалобы и сообщения администратору. Администратор может просматривать неотвеченные сообщения, отвечать пользователям и вести полную историю переписки с пагинацией. Бот поддерживает русский и английский языки, а также ограничивает количество активных сообщений от одного пользователя.

🌟 Ключевые возможности

👤 Для пользователей

  • 🌐 Мультиязычность (русский / английский)
  • 📩 Отправка трех типов сообщений: предложение, жалоба, простое сообщение
  • ⏳ Ограничение – одно активное сообщение (новое только после ответа администратора)

👨‍💻 Для администратора

  • 🔔 Просмотр неотвеченных сообщений
  • ✉️ Ответ на конкретные сообщения (прямо из бота)
  • 📜 История переписки с пагинацией
  • 🌐 Поддержка мультиязычности в интерфейсе админа

🛠 Технологический стек

Python 3.10+ python-telegram-bot 20.0+ SQLite3 asyncio Logging

🔧 Установка и настройка

📋 Требования

  • Python 3.10 или новее
  • Аккаунт Telegram
  • Бот, созданный через @BotFather

🔑 Получение токена и ID

  • Найдите @userinfobot в Telegram и получите свой chat ID
  • Через @BotFather создайте бота и получите токен

⚙️ Конфигурация

В файле config.py укажите:

YOUR_CHAT_ID = '123456789'        # ваш ID
TELEGRAM_BOT_TOKEN = 'токен:бота' # токен от BotFather

📦 Установка и запуск

pip install -r requirements.txt
python main.py

💡 Особенности реализации

  • Полностью асинхронная архитектура на python-telegram-bot (v20+)
  • Локализация через словари – легко добавить новые языки
  • База данных SQLite хранит все сообщения, диалоги и статусы
  • Логирование всех действий администратора и ошибок
  • Автоматическое создание БД и таблиц при первом запуске
