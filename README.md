## 📁 Структура проекта

```bash
portfolio/
├── css/
│   ├── binary.css                    # Стили для двоичного кода по бокам
│   ├── main_project_card.css         # Стили карточек проектов
│   ├── main.css                      # Основные стили (главная страница)
│   ├── project.css                   # Стили для страниц проектов
│   └── social_credit.css             # Стили для мемной анимации при лайке
├── js/
│   ├── components/
│   │   ├── achievements.js           # Рендер карточек достижений
│   │   ├── nav.js                    # Добавляет навигационное меню в проектов
│   │   ├── projects_cards.js         # Рендер карточек проектов
│   │   ├── screenshots_modal.js      # Работа с модальными окнами в карточках проектов
│   │   └── skills_align.js           # Выравнивает заголовки карточек навыков
│   ├── data/
│   │   ├── achievements_data.js      # Данные карточек моих достижений
│   │   └── projects_data.js          # Данные карточек моих проектов
│   └── utils/
│       ├── binary.js                 # Добавление двоичного кода по бокам страницы
│       └── update_year.js            # Автообновление года в футере
├── media/                            # Используемые медиафайлы
│   ├── achievements/
│   ├── icons/
│   ├── main/
│   ├── project_pizzeria/
│   └── sounds/
├── projects/
│   ├── project_ml.html               # Проект по изучению ML
│   ├── project_pizzeria.html         # Проект с сайтом пиццерии
│   └── project_telegram_bot.html     # Проект с телеграм ботом
├── index.html                        # Главная страница
├── LICENSE
├── README.en.md
└── README.md
```

## ✨ Особенности
- **Адаптивность** – корректное отображение на всех устройствах (от смартфонов до десктопов)
- **Модульность** – данные проектов и достижений вынесены в отдельные JS-файлы для легкого обновления
- **Безопасность** – экранирование HTML при вставке пользовательских данных (защита от XSS)
- **Модальное окно** – просмотр скриншотов проектов в полноразмерном режиме
- **Анимации** – плавное появление карточек, hover-эффекты, градиенты
